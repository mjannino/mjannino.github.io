Michael Jannino's Midterm Blackjack Project

I created my midterm project in front end technologies including css3, html5, and vanilla javascript. I avoided using any libraries for animations, layout configuration and the sort.

Each view has three files for the view due to the nature of front end technologies:

  • A "viewX.html" where the semantic HTML lives
  • A "viewX.css" where styling is done for the two views
  • A "viewX.js" where the true 'view' work is being done in the MVC design.

Both views share ONE model.js and controller.js, found in the root directry. Both views also share a "resource" folder.
